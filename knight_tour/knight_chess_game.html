Game Info: The purpose of the game is to travel each individual Square on the Chessboard using only the Knight piece. You are allowed to travel each square only once. The game is won only when you successful conquer each square on the chessboard. If you run out of squares to travel, but failed to conquer the whole chessboard the game is over.
